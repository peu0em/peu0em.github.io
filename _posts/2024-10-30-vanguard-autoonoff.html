뱅가드는 라이엇의 안티치트 프로그램입니다. 문제는 이게 시작 프로그램이어야 롤을 할 수 있는 것처럼 착각하기 쉽습니다. 시작프로그램에서 제외시키면 재부팅하라고 안내가 뜨니까요.1

그런데 사실 아닙니다. 라이엇 클라이언트 켜기 전에 뱅가드를 켜주기만 하면 돼요. 그러니까 롤을 할 때만 뱅가드를 켜고 아닐 때는 끄려면 이렇게 하면 됩니다.

  1. 뱅가드를 켠다.
  2. 라이엇 클라이언트를 켠다.
  3. 게임을 한다.
  4. 게임을 끈다.
  5. 라이엇 클라이언트를 끈다.
  6. 뱅가드 프로세스를 종료시킨다.

근데 이거 손으로 하기 귀찮잖아요. 그래서 제가 만들어 쓰는 배치파일 코드를 공유해드리려고 합니다. 사용법은 간단합니다. 내용을 메모장에 붙여넣고 .bat 확장자로 저장하면 됩니다. (혹시 글자가 깨지면 저장 화면에서 저장 버튼 옆에 있는 인코딩을 ANSI로 바꾸면 됩니다.)

@echo off rem 항상 관리자 권한으로 실행할 것! rem 뱅가드 위치 set vanguard="C:\Program Files\Riot Vanguard\vgtray.exe" rem 라이엇 클라이언트 위치 set riot="C:\Program Files\Riot Games\Riot Client\RiotClientServices.exe" rem ____________________________ echo 클라이언트 실행 중... start "" %vanguard% start "" %riot% echo 뱅가드를 끄려면 클라이언트를 종료한 후 아무 키나 누르세요. pause>nul taskkill /im RiotClientServices.exe /t /f taskkill /im vgc.exe /t /f taskkill /im vgtray.exe /t /f

사용 전에 해야 할 것 두 가지가 있습니다. 우선 이 파일을 그냥 쌩으로 쓰지 마시고 어디 다른 곳에 보관해둔 뒤에 바로가기를 만들어서 써 주세요. 그래야 바로가기 속성에서 관리자 권한으로 실행하는 옵션을 켤 수 있습니다. 또 저기 뱅가드 위치와 라이엇 클라이언트 위치를 수정해 주세요. 프로그램 설치 위치는 각자 다르니까요. 기본 위치는 둘 다 위와 같습니다. 그냥 거기 있는지 확인만 해주세요.

이제 라이엇 클라이언트 대신 저걸 실행하면 됩니다. 그러면 창이 안 꺼지고 계속 남아있는데요, 이걸 그냥 놔두고 게임을 한 다음에 게임 끄고 저 창에서 아무 키(글자 키 위주. 안 먹는 키가 있어요.)나 누르시면 뱅가드가 꺼지고 저 창도 꺼집니다.

1 사실 보편적으로 어떻게 알고 있는지는 모르지만, 스트리머 중에 롤 하려면 재부팅해야 한다고 알고 계신 분이 계셔서 이 글을 쓰게 됐습니당.
