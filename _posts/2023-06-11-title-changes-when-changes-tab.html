방법

<script> 태그를 사용할 수 있는 경우에만 가능합니다.
달리 말하면 <script>를 사용할 수만 있다면 가능하다..! 

window.addEventListener( "focus" , () => {
  document.title = "보고 있을 때";
});
window.addEventListener( "blur", () => {
  document.title = "전환됐을 때";
});

이 코드를 <script></script> 안에 넣으면 됩니다.
블로그처럼 원래 페이지에서 제공하는 제목이 있다면 먼저 변수로 지정해놓고 사용하세요. 

let 변수이름 = document.title;

위와 같이 미리 저장해둔 후에 원래의 제목을 사용할 부분에 사용하시면 됩니다.
